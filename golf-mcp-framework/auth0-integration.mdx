## Auth0 OAuth proxy configuration

Add Auth0 OAuth proxy configuration to your Golf MCP server's `auth.py`:

### Basic Auth0 configuration
```python
from golf.auth import configure_oauth_proxy, JWTAuthConfig

# Configure Auth0 OAuth proxy in your auth.py
configure_oauth_proxy(
    authorization_endpoint="https://your-tenant.us.auth0.com/authorize",
    token_endpoint="https://your-tenant.us.auth0.com/oauth/token", 
    client_id="your_auth0_client_id",
    client_secret="your_auth0_client_secret",
    base_url="https://mcp.example.com",
    scopes_supported=["read:resources", "openid", "profile"],
    environment="production",
    token_verifier_config=JWTAuthConfig(
        jwks_uri="https://your-tenant.us.auth0.com/.well-known/jwks.json",
        issuer="https://your-tenant.us.auth0.com/",
        audience="auth0-api-audience",
        required_scopes=["read:resources"]
    )
)
```

### Production Auth0 configuration with environment variables
```python
from golf.auth import configure_oauth_proxy, JWTAuthConfig
import os

# Recommended: use environment variables for Auth0 secrets
configure_oauth_proxy(
    authorization_endpoint=os.getenv("AUTH0_AUTHORIZATION_ENDPOINT"),
    token_endpoint=os.getenv("AUTH0_TOKEN_ENDPOINT"),
    client_id=os.getenv("AUTH0_CLIENT_ID"),
    client_secret=os.getenv("AUTH0_CLIENT_SECRET"),
    base_url=os.getenv("PROXY_BASE_URL"),
    scopes_supported=["read:resources", "openid", "profile"],
    environment=os.getenv("DEPLOYMENT_ENV", "production"),
    token_verifier_config=JWTAuthConfig(
        jwks_uri=os.getenv("AUTH0_JWKS_URI"),
        issuer=os.getenv("AUTH0_ISSUER"),
        audience=os.getenv("AUTH0_AUDIENCE"),
        required_scopes=["read:resources"]
    )
)
```

## Environment configuration

Set `environment` based on deployment:
- `"production"` - Production (secure, no debug info)
- `"development"` - Development (includes debug correlation IDs)
- `"local"` - Local development

## Required Auth0 environment variables for production

```bash
# Auth0 Application Settings (from your Auth0 Dashboard)
AUTH0_CLIENT_ID=your_auth0_client_id
AUTH0_CLIENT_SECRET=your_auth0_client_secret
AUTH0_AUTHORIZATION_ENDPOINT=https://your-tenant.us.auth0.com/authorize
AUTH0_TOKEN_ENDPOINT=https://your-tenant.us.auth0.com/oauth/token

# Your MCP Proxy Server Settings
PROXY_BASE_URL=https://mcp.example.com

# Auth0 JWT Verification Settings
AUTH0_JWKS_URI=https://your-tenant.us.auth0.com/.well-known/jwks.json
AUTH0_ISSUER=https://your-tenant.us.auth0.com/
AUTH0_AUDIENCE=auth0-api-audience
```

## Auth0 application setup

### Callback URL configuration

In your Auth0 Dashboard, configure your application with this callback URL:
```
{base_url}/oauth/callback
```

Example: `https://mcp.example.com/oauth/callback`

## Auth0 integration checklist

1. **Auth0 application setup**: Create Regular Web Application in Auth0 Dashboard
2. **Callback URL**: Configure `{base_url}/oauth/callback` in Auth0 application settings
3. **Environment variables**: Set all required Auth0 env vars from your application settings
4. **HTTPS**: Ensure base_url uses HTTPS in production (required by Auth0)
5. **Scopes**: Configure appropriate scopes (e.g., `read:resources`, `openid`, `profile`)
6. **JWT verification**: Ensure JWKS URI points to your Auth0 tenant's well-known endpoint
7. **Test**: Verify complete OAuth flow works end-to-end with Auth0
